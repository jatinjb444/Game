What is a Subset?

A subset is a set in which every element of the subset is also an element of another set, called the superset.

How to Play

  1. Select the subsets from the 1-15 numbers displayed in the game.
  2. Each round, you need to find five valid subsets within the time limit.
  3. Earn 50 points for completing each round. The game ends after reaching 200 points!
